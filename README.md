# Wayne 0.3

Buscador funcional de cursos gratuitos.

## Estrutura

- `index.html` — página principal
- `css/style.css` — visual do Wayne
- `js/app.js` — busca, filtros e favoritos
- `data/cursos.json` — base inicial de cursos
- `manifest.json` — base para PWA

## Como publicar no GitHub pelo celular

Envie estes arquivos e pastas para o repositório:

- index.html
- css/
- js/
- data/
- manifest.json

O Netlify vai atualizar automaticamente.
