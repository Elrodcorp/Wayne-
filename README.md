# Wayne 1.0 — Portfólio funcional

Wayne é um buscador inteligente de cursos gratuitos, criado como primeiro produto da Elrod Corp.

## O que já funciona
- Busca por área, profissão ou habilidade
- Filtros por área, nível e certificado
- Cards de cursos
- Wayne Score
- Favoritos salvos no navegador
- Trilha "Do curso ao mercado"
- Layout responsivo
- Projeto pronto para Netlify

## Netlify
O arquivo `netlify.toml` já configura:
- Build command: `npm run build`
- Publish directory: `dist`
