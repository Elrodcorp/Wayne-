# Wayne

Wayne é uma plataforma de inteligência de carreira: o usuário descreve onde quer chegar e recebe trilhas, cursos, habilidades e oportunidades.

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Supabase
- Cloudflare Pages

## Como rodar localmente

```bash
npm install
npm run dev
```

## Como publicar no Cloudflare Pages

Build command:

```bash
npm run build
```

Build output directory:

```bash
dist
```
