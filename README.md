# Wayne Visual Dashboard Pro

Projeto estático para GitHub + Netlify.

## Como publicar
1. Envie estes arquivos para o repositório:
   - index.html
   - style.css
   - app.js
   - data.js

2. No Netlify:
   - Se já estiver conectado ao GitHub, basta dar commit.
   - Não precisa build.
   - Publish directory: deixe em branco ou use a raiz do projeto.

## Como testar localmente
Abra o arquivo `index.html` no navegador.
