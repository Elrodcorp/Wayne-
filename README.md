# Wayne 0.4 — React + Vite

Versão em framework do Wayne, pronta para Netlify.

## Configuração no Netlify

Build command:
npm run build

Publish directory:
dist

## Arquivos principais

- `src/main.jsx` — aplicação React
- `src/style.css` — layout premium
- `src/courses.js` — base inicial de cursos
- `package.json` — dependências e scripts
